我们自定义的udp数据包的优先级是通过在数据包中的一个字段值来实现的。'0'表示最低优先级，'1'表示低优先级，'2'表示高优先级。模拟网络环境时，在处理数据包时，我们首先解析出这个优先级字段，然后根据优先级和对应的概率来决定是否丢弃该数据包。
以下为具体实现逻辑，通过在clumsy 的drop.c中 添加以下代码来实现：
```C++
PacketNode *pac = head->next;
// chance in range of [0, 10000]

PWINDIVERT_UDPHDR udp = NULL;
char *data = NULL;
UINT dataLen = 0;

WinDivertHelperParsePacket(
    pac->packet,
    pac->packetLen,
    NULL,
    NULL,
    NULL,
    NULL,
    NULL,
    NULL,
    &udp,
    (PVOID *)&data,
    &dataLen,
    NULL,
    NULL);

    
char PRI;
for (int i = 1; i <= 40; i++)
    if (data[i] == '\"' && data[i + 1] == 'v' && data[i + 2] == '\"' && data[i + 3] == ':')
    {
        PRI = data[i + 4];
        LOG("Find Priority of this udp data:   %c", PRI);
        break;
    }

if (checkDirection(pac->addr.Outbound, dropInbound, dropOutbound) && calcChance(chance) && udp != NULL && PRI == '0')
{ // 我们自己定义的udp数据的优先级， 2 高优先级 1 低优先级 0 最低优先级
    LOG("dropped with chance %.1f%%, direction %s",
        chance / 100.0, pac->addr.Outbound ? "OUTBOUND" : "INBOUND");
    freeNode(popNode(pac));
    ++dropped;
}
else if (checkDirection(pac->addr.Outbound, dropInbound, dropOutbound) && calcChance(chance / 2) && udp != NULL && PRI == '1')
{
    LOG("dropped with chance %.1f%%, direction %s",
        chance / 200.0, pac->addr.Outbound ? "OUTBOUND" : "INBOUND");
    freeNode(popNode(pac));
    ++dropped;
}
else
{
    head = head->next;
}
```
